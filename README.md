# Video Player MMT

Một video player hiện đại sử dụng ArtPlayer.js và Video.js với nhiều tính năng cao cấp.

## Tính năng

- 🎬 Phát video với ArtPlayer và Video.js
- 🔊 Điều chỉnh âm lượng với indicator
- 💡 Điều chỉnh độ sáng (brightness)
- ⏭️ Tua nhanh/tua chậm 5s
- 🖼️ Xem trước khi tua (seek preview)
- 📱 Hỗ trợ mobile UI
- ⌨️ Phím tắt bàn phím
- 💾 Lưu vị trí xem tiếp
- 🎯PIP (Picture in Picture)
- 📷 Chụp ảnh màn hình
- 🔄 Tự động lưu và khôi phục tiến độ

## Files

- `mmt video player.js` - Video player chính với Video.js
- `artplayer mmt.js` - ArtPlayer configuration
- `artplayer original.js` - ArtPlayer library (original)

## Usage

Mở file HTML trong trình duyệt để xem video.
